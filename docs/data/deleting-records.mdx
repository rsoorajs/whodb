# Deleting Records

WhoDB shows delete actions only when the connected source and selected object expose delete support.

## Delete A Row

<Steps>
<Step title="Open Row Actions">
Right-click the row you want to delete.
</Step>
<Step title="Select Delete Row">
Choose **Delete Row** from the context menu.
</Step>
<Step title="Confirm">
Current CE code opens a confirmation dialog before deletion.
</Step>
<Step title="Review Result">
WhoDB deletes the row through the backend plugin, shows a toast, and refreshes the grid.
</Step>
</Steps>

## Delete Selected Rows

Use row checkboxes to select multiple loaded rows. When any rows are selected, the delete action operates on the selected rows.

## Keyboard Delete

When table shortcuts are active, focus a row and press:

- `Mod + Delete`
- `Mod + Backspace`

Both open the same confirmation flow.

<Warning>
Deletes are permanent in the connected database. Use filters carefully and verify the selected rows before confirming.
</Warning>
