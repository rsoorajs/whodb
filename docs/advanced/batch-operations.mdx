# Batch Operations

The table grid supports row selection and selection-based actions for sources that expose those capabilities.

![Row Selection Single](/images/44-table-row-selection-single.png)

## Select Rows

Use any of these current selection methods:

- Click a row checkbox.
- Use the header checkbox to select or deselect the visible page.
- Focus a row and press `Space`.
- Use `Shift + Arrow Up/Down` to extend selection.
- Use `Cmd/Ctrl + A` to select all rows in the current view.
- Use the row context menu action **Select Row**.

![Select Row From Context Menu](/images/45-context-menu-select-row.png)

Use the header checkbox, row checkboxes, shortcuts, and context menus to select or clear rows.

## Export Selected Rows

After selecting rows, use the selected-row export action.

![Export Selected Rows](/images/46-export-selected-rows-dialog.png)

The export starts from the selected rows you chose in the grid.

## Edit And Delete

If the connected source allows writes:

- `Enter` or `Cmd/Ctrl + E` opens edit for the focused row.
- `Cmd/Ctrl + Delete` or `Cmd/Ctrl + Backspace` starts delete for the focused row.
- The row context menu also exposes **Edit Row** and **Delete Row** when supported.

Delete actions show a confirmation dialog before the mutation is submitted.

## Context Menus

Right-click a grid row to open the row context menu.

![Context Menu](/images/13-data-view-context-menu.png)

Header context menus expose table-level actions such as refresh, export, select/deselect all visible rows, and mock data when supported.

![Table Header Context Menu](/images/37-table-header-context-menu.png)
