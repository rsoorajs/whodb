# Data Exploration Workflow

This workflow follows the current WhoDB CE interface.

## 1. Choose The Object

Open **Storage Units** and use search to find the object you need.

![Storage Unit List](/images/05-storage-unit-list.png)

Use **Describe** to inspect structure and metadata.

![Explore Users Table](/images/07-explore-users-table.png)

Use **Data** to open rows.

![Data View Users Table](/images/09-data-view-users-table.png)

## 2. Inspect Rows

The data grid loads rows automatically. Use page size and pagination to control how much data is visible.

![Data View Table Content](/images/10-data-view-table-content.png)

![Page Size Dropdown](/images/15-data-view-page-size-dropdown.png)

## 3. Search Visible Rows

The Search field searches the rows currently loaded in the grid and highlights/scrolls matches.

![Search Highlight](/images/19-data-view-search-highlight.png)

Use WHERE conditions when you want to reduce the result count.

## 4. Add WHERE Conditions

Use WHERE conditions to filter server-side.

<Steps>
<Step title="Open Condition UI">
Click **Add** beside **Where condition**.
</Step>
<Step title="Build Condition">
Select field, operator, and value.
</Step>
<Step title="Add Condition">
Add the condition to the list.
</Step>
<Step title="Run Query">
Click **Query** to reload the grid.
</Step>
</Steps>

![Where Conditions Popover](/images/16-data-view-where-conditions-popover.png)

![Multiple Conditions](/images/39-data-view-multiple-conditions.png)

## 5. Sort Columns

Click a column header to sort. Shift-click additional headers for multi-column sorting.

![Sorted Column](/images/38-data-view-sorted-column.png)

## 6. Use Scratchpad For Custom Questions

Open **Scratchpad** from the sidebar or the Storage Units shortcut and run SQL for custom analysis.

![Scratchpad Code Editor](/images/28-scratchpad-code-editor.png)

```sql
SELECT status, COUNT(*) AS total
FROM users
GROUP BY status
ORDER BY total DESC;
```

![Scratchpad Query Results](/images/29-scratchpad-query-results.png)

## 7. Export When Needed

Use export from the grid after applying filters or selecting rows.

![Export Dialog](/images/20-data-view-export-dialog.png)

Current formats are CSV, Excel, and JSON Lines for records from sources like MongoDB when supported.
